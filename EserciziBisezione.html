1. Dalle duplicazione alla bisezione

1.1 Le formule di duplicazione (ripasso)

Ricordiamo le formule di duplicazione del coseno:

Formule di duplicazione
\[ \cos(2\alpha) = \cos^2\alpha - \sen^2\alpha \] Usando l'identit√† fondamentale \(\sen^2\alpha + \cos^2\alpha = 1\), si ottengono due forme equivalenti: \[ \cos(2\alpha) = 2\cos^2\alpha - 1 \qquad\qquad \cos(2\alpha) = 1 - 2\sen^2\alpha \]

1.2 La sostituzione chiave

L'idea √® semplice: se poniamo \(\alpha = \dfrac{\theta}{2}\), allora \(2\alpha = \theta\). Sostituiamo nelle due forme:

Sostituzione \(\alpha = \theta/2\)
\[ \cos\theta = 2\cos^2\!\frac{\theta}{2} - 1 \qquad\qquad \cos\theta = 1 - 2\sen^2\!\frac{\theta}{2} \]

1.3 Inversione: le formule di bisezione

Ora isoliamo \(\cos^2\!\dfrac{\theta}{2}\) dalla prima e \(\sen^2\!\dfrac{\theta}{2}\) dalla seconda:

1
Dalla prima forma:
\(\cos\theta = 2\cos^2\!\dfrac{\theta}{2} - 1\)  ‚üπ  \(2\cos^2\!\dfrac{\theta}{2} = 1 + \cos\theta\)  ‚üπ  \(\cos^2\!\dfrac{\theta}{2} = \dfrac{1+\cos\theta}{2}\)
2
Dalla seconda forma:
\(\cos\theta = 1 - 2\sen^2\!\dfrac{\theta}{2}\)  ‚üπ  \(2\sen^2\!\dfrac{\theta}{2} = 1 - \cos\theta\)  ‚üπ  \(\sen^2\!\dfrac{\theta}{2} = \dfrac{1-\cos\theta}{2}\)
Formule di bisezione
\[ \cos^2\!\frac{\theta}{2} = \frac{1+\cos\theta}{2} \qquad\qquad \sen^2\!\frac{\theta}{2} = \frac{1-\cos\theta}{2} \] E, dividendo membro a membro: \[ \tan^2\!\frac{\theta}{2} = \frac{1-\cos\theta}{1+\cos\theta} \]
‚ö†Ô∏è Attenzione al segno!

Le formule danno il quadrato di seno e coseno. Quando estraiamo la radice quadrata, il segno (¬±) dipende dal quadrante in cui si trova l'angolo \(\theta/2\):

\[ \cos\frac{\theta}{2} = \pm\sqrt{\frac{1+\cos\theta}{2}} \qquad\qquad \sen\frac{\theta}{2} = \pm\sqrt{\frac{1-\cos\theta}{2}} \]

2. Esempio guidato: calcolo di \(\cos\dfrac{9}{8}\pi\)

Vogliamo calcolare il valore esatto di \(\cos\dfrac{9}{8}\pi\). Procediamo passo per passo.

1
Riconoscere l'angolo come met√† di un angolo noto

Osserviamo che:

\[\frac{9}{8}\pi = \frac{1}{2}\cdot\frac{9}{4}\pi\]

Quindi \(\dfrac{9}{8}\pi\) √® la met√† di \(\dfrac{9}{4}\pi\). Posto \(\theta = \dfrac{9}{4}\pi\), dobbiamo calcolare \(\cos\dfrac{\theta}{2}\).

2
Calcolare \(\cos\theta = \cos\dfrac{9}{4}\pi\)

L'angolo \(\dfrac{9}{4}\pi = 2\pi + \dfrac{1}{4}\pi\), quindi √® coterminale con \(\dfrac{\pi}{4}\):

\[\cos\frac{9}{4}\pi = \cos\frac{\pi}{4} = \frac{\sqrt{2}}{2}\]
3
Applicare la formula di bisezione

Sostituiamo nella formula:

\[\cos^2\!\frac{\theta}{2} = \frac{1+\cos\theta}{2} = \frac{1 + \dfrac{\sqrt{2}}{2}}{2} = \frac{\dfrac{2+\sqrt{2}}{2}}{2} = \frac{2+\sqrt{2}}{4}\]
4
Determinare il segno

L'angolo \(\dfrac{9}{8}\pi\) √® compreso tra \(\pi\) e \(\dfrac{3}{2}\pi\), quindi si trova nel terzo quadrante, dove il coseno √® negativo.

Quindi scegliamo il segno meno davanti alla radice.

5
Estrarre la radice e concludere
\[\cos\frac{9}{8}\pi = -\sqrt{\frac{2+\sqrt{2}}{4}} = -\frac{\sqrt{2+\sqrt{2}}}{2}\]
üìù Schema riassuntivo del procedimento

Per calcolare \(\cos\dfrac{\alpha}{2}\) o \(\sen\dfrac{\alpha}{2}\):

(a) Scrivi l'angolo come met√† di un angolo \(\theta\), cio√® trova \(\theta = 2\cdot\text{(angolo dato)}\).

(b) Calcola \(\cos\theta\) (riconducendoti, se serve, a un angolo noto).

(c) Sostituisci nella formula di bisezione.

(d) Determina il segno guardando in quale quadrante si trova l'angolo dato.

(e) Estrai la radice quadrata.

Gruppo A ‚Äî Applicazione diretta delle formule

Calcola i seguenti valori esatti usando le formule di bisezione.

Stesso \(\theta = \dfrac{\pi}{4}\), usiamo la formula per il seno:

\(\sen^2\dfrac{\pi}{8} = \dfrac{1 - \frac{\sqrt{2}}{2}}{2} = \dfrac{2-\sqrt{2}}{4}\)

\(\dfrac{\pi}{8}\) √® nel primo quadrante, seno positivo:

\(\cos^2\dfrac{\pi}{12} = \dfrac{1 + \frac{\sqrt{3}}{2}}{2} = \dfrac{2+\sqrt{3}}{4}\)

A5. Calcola \(\tan\dfrac{\pi}{8}\).

Con la formula diretta, \(\theta = \dfrac{\pi}{4}\), \(\cos\theta = \dfrac{\sqrt{2}}{2}\):

\(\tan^2\dfrac{\pi}{8} = \dfrac{1 - \frac{\sqrt{2}}{2}}{1 + \frac{\sqrt{2}}{2}} = \dfrac{2 - \sqrt{2}}{2 + \sqrt{2}}\)

Razionalizziamo:

\(\dfrac{2 - \sqrt{2}}{2 + \sqrt{2}} \cdot \dfrac{2 - \sqrt{2}}{2 - \sqrt{2}} = \dfrac{(2-\sqrt{2})^2}{4-2} = \dfrac{(2-\sqrt{2})^2}{2}\)

Primo quadrante, tangente positiva:

\[\tan\frac{\pi}{8} = \frac{2-\sqrt{2}}{\sqrt{2}} = \sqrt{2}-1\]

Gruppo B ‚Äî Determinazione del segno

In questi esercizi √® fondamentale determinare correttamente il quadrante.

B1. Calcola \(\sen\dfrac{9}{8}\pi\).
Mostra soluzione

\(\dfrac{9}{8}\pi = \dfrac{1}{2}\cdot\dfrac{9}{4}\pi\), \(\theta = \dfrac{9}{4}\pi\), \(\cos\theta = \cos\dfrac{\pi}{4} = \dfrac{\sqrt{2}}{2}\).

\(\dfrac{5}{8}\pi \approx 0.625\pi\) √® nel secondo quadrante, coseno negativo:

\[\cos\frac{5}{8}\pi = -\frac{\sqrt{2-\sqrt{2}}}{2}\]
B3. Calcola \(\sen\dfrac{5}{8}\pi\).
Mostra soluzione

\(\theta = \dfrac{5}{4}\pi\), \(\cos\theta = -\dfrac{\sqrt{2}}{2}\):

\(\sen^2\dfrac{5}{8}\pi = \dfrac{1 - \left(-\frac{\sqrt{2}}{2}\right)}{2} = \dfrac{2+\sqrt{2}}{4}\)

\(\dfrac{5}{8}\pi\) √® nel secondo quadrante, seno positivo:

\[\sen\frac{5}{8}\pi = \frac{\sqrt{2+\sqrt{2}}}{2}\]
B4. Calcola \(\cos\dfrac{11}{8}\pi\).
Mostra soluzione

\(\dfrac{11}{8}\pi = \dfrac{1}{2}\cdot\dfrac{11}{4}\pi\), \(\theta = \dfrac{11}{4}\pi = 2\pi + \dfrac{3}{4}\pi\), \(\cos\theta = \cos\dfrac{3}{4}\pi = -\dfrac{\sqrt{2}}{2}\).

\(\cos^2\dfrac{11}{8}\pi = \dfrac{1 - \frac{\sqrt{2}}{2}}{2} = \dfrac{2-\sqrt{2}}{4}\)

\(\dfrac{11}{8}\pi \approx 1.375\pi\) √® nel terzo quadrante, coseno negativo:

\[\cos\frac{11}{8}\pi = -\frac{\sqrt{2-\sqrt{2}}}{2}\]
B5. Calcola \(\sen\dfrac{13}{8}\pi\).
Mostra soluzione

\(\dfrac{13}{8}\pi = \dfrac{1}{2}\cdot\dfrac{13}{4}\pi\), \(\theta = \dfrac{13}{4}\pi = 2\pi + \dfrac{5}{4}\pi\), \(\cos\theta = \cos\dfrac{5}{4}\pi = -\dfrac{\sqrt{2}}{2}\).

\(\dfrac{13}{8}\pi \approx 1.625\pi\) √® nel quarto quadrante, seno negativo:

\[\sen\frac{13}{8}\pi = -\frac{\sqrt{2+\sqrt{2}}}{2}\]

Mostra soluzione

\(\theta = \dfrac{5}{6}\pi\), \(\cos\theta = -\dfrac{\sqrt{3}}{2}\):

Gruppo D ‚Äî Esercizi con parametro

In questi esercizi √® dato il valore di una funzione goniometrica di \(\theta\) e bisogna trovare \(\cos\dfrac{\theta}{2}\), \(\sen\dfrac{\theta}{2}\) o \(\tan\dfrac{\theta}{2}\).

D1. Sapendo che \(\cos\theta = \dfrac{3}{5}\) e che \(0 < \theta < \dfrac{\pi}{2}\), calcola \(\cos\dfrac{\theta}{2}\) e \(\sen\dfrac{\theta}{2}\).
Mostra soluzione

Se \(0 < \theta < \dfrac{\pi}{2}\), allora \(0 < \dfrac{\theta}{2} < \dfrac{\pi}{4}\) (primo quadrante): sia seno che coseno sono positivi.

\(\cos\dfrac{\theta}{2} = \sqrt{\dfrac{1 + \frac{3}{5}}{2}} = \sqrt{\dfrac{\frac{8}{5}}{2}} = \sqrt{\dfrac{4}{5}} = \dfrac{2}{\sqrt{5}} = \dfrac{2\sqrt{5}}{5}\)

\(\sen\dfrac{\theta}{2} = \sqrt{\dfrac{1 - \frac{3}{5}}{2}} = \sqrt{\dfrac{\frac{2}{5}}{2}} = \sqrt{\dfrac{1}{5}} = \dfrac{1}{\sqrt{5}} = \dfrac{\sqrt{5}}{5}\)

D2. Sapendo che \(\cos\theta = -\dfrac{1}{3}\) e che \(\pi < \theta < \dfrac{3}{2}\pi\), calcola \(\cos\dfrac{\theta}{2}\) e \(\sen\dfrac{\theta}{2}\).
Mostra soluzione

Se \(\pi < \theta < \dfrac{3}{2}\pi\), allora \(\dfrac{\pi}{2} < \dfrac{\theta}{2} < \dfrac{3}{4}\pi\) (secondo quadrante): seno positivo, coseno negativo.

\(\cos^2\dfrac{\theta}{2} = \dfrac{1 - \frac{1}{3}}{2} = \dfrac{\frac{2}{3}}{2} = \dfrac{1}{3}\)  ‚üπ  \(\cos\dfrac{\theta}{2} = -\dfrac{1}{\sqrt{3}} = -\dfrac{\sqrt{3}}{3}\)

\(\sen^2\dfrac{\theta}{2} = \dfrac{1 + \frac{1}{3}}{2} = \dfrac{\frac{4}{3}}{2} = \dfrac{2}{3}\)  ‚üπ  \(\sen\dfrac{\theta}{2} = \sqrt{\dfrac{2}{3}} = \dfrac{\sqrt{6}}{3}\)

D3. Sapendo che \(\cos\theta = \dfrac{7}{25}\) e che \(\dfrac{3}{2}\pi < \theta < 2\pi\), calcola \(\tan\dfrac{\theta}{2}\).
Mostra soluzione

Se \(\dfrac{3}{2}\pi < \theta < 2\pi\), allora \(\dfrac{3}{4}\pi < \dfrac{\theta}{2} < \pi\) (secondo quadrante): tangente negativa.

\(\tan^2\dfrac{\theta}{2} = \dfrac{1 - \frac{7}{25}}{1 + \frac{7}{25}} = \dfrac{\frac{18}{25}}{\frac{32}{25}} = \dfrac{18}{32} = \dfrac{9}{16}\)

\(\tan\dfrac{\theta}{2} = -\dfrac{3}{4}\)

D4. Sapendo che \(\sen\theta = \dfrac{4}{5}\) e che \(\dfrac{\pi}{2} < \theta < \pi\), calcola \(\cos\dfrac{\theta}{2}\).
Suggerimento: prima ricava \(\cos\theta\) dall'identit√† fondamentale, poi usa la bisezione.
Mostra soluzione

Nel secondo quadrante: \(\cos\theta = -\sqrt{1 - \frac{16}{25}} = -\dfrac{3}{5}\).

Se \(\dfrac{\pi}{2} < \theta < \pi\), allora \(\dfrac{\pi}{4} < \dfrac{\theta}{2} < \dfrac{\pi}{2}\) (primo quadrante): coseno positivo.

\(\cos\dfrac{\theta}{2} = \sqrt{\dfrac{1 - \frac{3}{5}}{2}} = \sqrt{\dfrac{1}{5}} = \dfrac{\sqrt{5}}{5}\)

Gruppo E ‚Äî Semplificazione di espressioni

E1. Semplifica: \(\dfrac{1 - \cos(2\alpha)}{1 + \cos(2\alpha)}\).
Mostra soluzione

Pensiamo a \(2\alpha\) come l'angolo \(\theta\) e ad \(\alpha\) come \(\theta/2\). Allora:

Numeratore: \(1 - \cos(2\alpha) = 2\sen^2\alpha\)

Denominatore: \(1 + \cos(2\alpha) = 2\cos^2\alpha\)

\[\frac{1 - \cos(2\alpha)}{1 + \cos(2\alpha)} = \frac{2\sen^2\alpha}{2\cos^2\alpha} = \tan^2\alpha\]
E2. Semplifica: \(\sqrt{\dfrac{1 + \cos(6x)}{2}}\), sapendo che \(0 < x < \dfrac{\pi}{6}\).
Mostra soluzione

\(\dfrac{1 + \cos(6x)}{2} = \cos^2(3x)\) (formula di bisezione con \(\theta = 6x\)).

Se \(0 < x < \dfrac{\pi}{6}\), allora \(0 < 3x < \dfrac{\pi}{2}\): coseno positivo.

\[\sqrt{\frac{1 + \cos(6x)}{2}} = |\cos(3x)| = \cos(3x)\]
E3. Semplifica: \(\dfrac{\sen\theta}{1 + \cos\theta}\).
Suggerimento: usa le formule di bisezione al numeratore e al denominatore.
Mostra soluzione

Dalle formule di duplicazione/bisezione:

\(\sen\theta = 2\sen\dfrac{\theta}{2}\cos\dfrac{\theta}{2}\) (duplicazione del seno)

\(1 + \cos\theta = 2\cos^2\dfrac{\theta}{2}\)

\[\frac{\sen\theta}{1+\cos\theta} = \frac{2\sen\frac{\theta}{2}\cos\frac{\theta}{2}}{2\cos^2\frac{\theta}{2}} = \frac{\sen\frac{\theta}{2}}{\cos\frac{\theta}{2}} = \tan\frac{\theta}{2}\]
E4. Verifica l'identit√†: \(\dfrac{1 - \cos\theta}{\sen\theta} = \tan\dfrac{\theta}{2}\).
Mostra soluzione

\(1 - \cos\theta = 2\sen^2\dfrac{\theta}{2}\)

\(\sen\theta = 2\sen\dfrac{\theta}{2}\cos\dfrac{\theta}{2}\)

\[\frac{1-\cos\theta}{\sen\theta} = \frac{2\sen^2\frac{\theta}{2}}{2\sen\frac{\theta}{2}\cos\frac{\theta}{2}} = \frac{\sen\frac{\theta}{2}}{\cos\frac{\theta}{2}} = \tan\frac{\theta}{2} \quad\checkmark\]
